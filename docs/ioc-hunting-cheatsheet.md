# IOC Hunting Cheat Sheet

## File Indicators

- SHA256 hashes
- MD5 hashes
- Suspicious executables
- Unexpected DLLs

## Network Indicators

- Malicious IP addresses
- Suspicious domains
- Unusual DNS requests
- Beaconing traffic

## Email Indicators

- Spoofed sender
- Malicious attachments
- Suspicious URLs
- Display name impersonation

## Endpoint Indicators

- New scheduled tasks
- Suspicious services
- Registry Run keys
- PowerShell encoded commands

## Log Sources

- Windows Event Logs
- Sysmon
- Splunk
- EDR telemetry
- Firewall logs
- Proxy logs

## MITRE ATT&CK References

- Initial Access
- Execution
- Persistence
- Privilege Escalation
- Defense Evasion
- Credential Access
- Discovery
- Lateral Movement
- Collection
- Exfiltration
